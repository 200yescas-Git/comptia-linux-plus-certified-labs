# CompTIA Linux+ 

Respositorio dedicado a mis prácticas,laboratorios y temas de estudios para la preparación y formalización de mis conocimientos GNU/Linux en la certificación de CompTIA Linux+.

## Objetivos
- Fortalezer habiliddades de administración es las diferentes familias de Linux
- Aprender Automatización con Bash
- Comprender la seguridad y redes de Linux
- Prepararme para entornos reales de Sysadmin 
- Aplicar la seguridad en los sistemas para mantener seguro nuestro sistema

## Temas Estudiados
- Estructura del sistema
- Comandos del sistema linux
- Gestión de usuarios y permisos para escalar privilegios
- Administración de procesos
- Networking 
- Servicios
- Shell Scripting
- Gestión de paquetes
- logs del sistema

## Herramientas
- VMware Workstation
- Bash
- SSH
- Git
- Github

- En vía de desarrollo
